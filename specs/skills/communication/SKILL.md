# Communication

**Step 1 — Announce**

> *Using: communication.*

**Step 2 — Route to the right skill**

| Skill | When to use |
|---|---|
| `notify` | Sending a Telegram or email notification |
| `send-email` | Sending a full email via Gmail on Daniel's behalf |
| `send-message-to-mind` | Sending an async message to another mind via the broker |
| `moderate` | Moderating a group conversation across multiple minds |
| `x-search` | Searching X (Twitter) for tweets or thread replies |
| `x-ai-lurker` | Fetching top AI threads on X for a daily digest |
